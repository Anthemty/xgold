# About GOX

## Our Mission

GOX aims to democratize access to gold investment by combining the stability of physical gold with the innovation of blockchain technology. We bridge traditional finance and decentralized finance (DeFi) to make gold investment accessible to everyone, everywhere.

## Vision

To become the world's leading decentralized exchange for gold-backed digital assets and Real World Assets (RWA), providing:
- **Transparency**: Real-time on-chain verification of all transactions
- **Accessibility**: 24/7 global access to gold markets
- **Security**: Non-custodial solutions with audited smart contracts
- **Efficiency**: Instant settlement with minimal fees

## What Makes GOX Different?

### Specialized Gold DEX
Unlike general-purpose DEXs, GOX is specifically optimized for gold-backed assets:
- Real-time gold spot price integration
- Historical price data from 1978
- Premium/discount tracking for tokenized gold
- Specialized liquidity pools for gold tokens

### RWA Focus
We support the entire ecosystem of Real World Assets:
- Physical gold tokenization
- Detection, custody & audit services
- RWA issuance & listing support
- Derivatives & OTC trading

### Multi-Chain Support
GOX operates on multiple blockchain networks:
- **Ethereum**: Primary network for PAXG and XAUT
- **Arbitrum**: Low-fee Layer 2 solution
- **Polygon**: Fast and scalable transactions

## Platform Architecture

### Frontend Layer
- Next.js 15 with App Router
- React 18 with TypeScript
- TailwindCSS for responsive UI
- Reown AppKit for wallet integration

### Data Layer
- **Supabase**: PostgreSQL database with real-time subscriptions
- **Metals.dev API**: Gold spot price data
- **CoinGecko API**: Token market data
- **Blockscout API**: On-chain holder count

### Smart Contracts
- ERC-20 token standards
- Uniswap V2/V3 liquidity pools
- Custom staking contracts
- Audited by leading security firms

### Key Integrations

```typescript
// Gold Price Feed
const goldPrice = await fetchGoldPrice();
// Returns: { price: 2645.50, timestamp: "2025-12-11T00:00:00Z" }

// Token Data
const paxgData = await fetchTokenPrice('PAXG');
// Returns: { price: 2646.12, volume_24h: 2847300, ... }

// Real-time Updates via Supabase
supabase
  .channel('gold-price-updates')
  .on('postgres_changes', { table: 'market_data' }, handler)
  .subscribe();
```
